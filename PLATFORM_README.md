# AI Motion Comic Studio - 智能动态漫创作平台

## 📖 平台简介
**AI Motion Comic Studio** 是一个全流程、高度内聚的 AI 内容创作平台，专为创作者设计。它打破了传统工具的割裂感，将 **剧本创作**、**资产管理**、**分镜绘制** 与 **视频生成** 无缝整合，让创作者能够在一个工作流中完成从灵感到动态成片的完整创作。

## 🚀 核心工作流

平台采用 **线性推进 + 资产复用** 的内聚工作模式：

1.  **✍️ 剧本创作 (Script Creation)**
    *   **灵感扩写**：输入一句话创意，AI (DeepSeek) 自动扩写为包含场景、运镜、角色的专业分镜脚本。
    *   **格式化输出**：自动将文本转化为结构化的分镜数据，无缝流转至下一环节。

2.  **📦 资产管理 (Asset Consistency)**
    *   **统一资产库**：集中管理角色（Character）与场景（Scene）。
    *   **一致性控制**：上传或生成角色三视图/参考图，所有后续分镜生成均自动调用该资产，确保角色长相不崩坏。

3.  **🎨 关键帧生成 (Keyframe Studio)**
    *   **多模型集成**：内置 Flux、Jimeng (即梦)、SDXL 等顶级绘画模型。
    *   **图生图/文生图**：自动读取剧本描述与资产参考图，一键生成分镜的起始帧与结束帧。
    *   **局部重绘**：内置 Inpainting 工具，精准修改画面瑕疵。

4.  **🎬 视频生成 (Video Generation)**
    *   **首尾帧驱动**：利用已确认的起始帧与结束帧，驱动生成高质量过场视频。
    *   **Doubao Seedance**：接入字节跳动最新的视频生成模型，支持大幅度动作与运镜。

## ✨ 平台亮点

*   **高度内聚**：数据不落地，剧本直接变分镜，分镜直接变视频。
*   **一致性保障**：基于资产库的生成逻辑，最大程度解决 AI 创作中“换镜头即换脸”的痛点。
*   **全链路日志**：详细记录每一次 API 调用（剧本、生图、视频），方便回溯与成本控制。
*   **可视化交互**：所见即所得的拖拽式看板，像搭积木一样做动画。

## 🛠️ 快速开始

1.  **配置密钥**：在 `.env.local` 中配置 DeepSeek、Volcengine (字节)、Fal.ai 等密钥。
2.  **启动服务**：`npm run dev`
3.  **开始创作**：访问 `http://localhost:3000`，从“剧本创作”开始你的第一部 AI 动画。
